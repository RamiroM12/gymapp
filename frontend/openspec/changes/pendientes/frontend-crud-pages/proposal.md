# Propuesta: Implementar páginas de Pagos, Membresías y Tipos de Membresía

## Intención
Agregar las funcionalidades faltantes en el frontend para gestionar:
1. **Tipos de Membresía** - CRUD completo (crear, listar, editar, eliminar)
2. **Membresías** - Listar membresías de clientes, comprar nueva membresía
3. **Pagos** - Listar pagos, registrar nuevo pago

## Alcance
- Crear 3 nuevas páginas en el frontend
- Conectar con los endpoints existentes del backend
- Mantener consistencia con el diseño actual

## Enfoque
- Usar la misma estructura de componentes existente (Cards, Inputs, Buttons)
- Implementar filtros de búsqueda donde sea necesario
- Seguir patrones de código ya establecidos en el proyecto
